# Muhamad Raihan - Portfolio Website

Website portofolio profesional untuk Muhamad Raihan (Full Stack Developer | Mobile Apps Developer | Web Developer).
Desain modern menggunakan konsep Glassmorphism, animasi mulus, dan 100% responsif di berbagai perangkat.

## Struktur Folder

```text
portfolio/
├── index.html       # File utama (Single Page Application layout)
├── style.css        # Semua styling CSS (termasuk responsif)
├── script.js        # Semua logika JavaScript (animasi, navbar, filtering)
├── assets/          # Semua aset statis
│   ├── images/      # Gambar profil dan ilustrasi
│   ├── projects/    # Tangkapan layar proyek
│   ├── certificates/# Tangkapan layar sertifikat
│   └── icons/       # Ikon lokal (jika ada)
└── README.md        # Dokumentasi proyek
```

## Fitur Utama
- **Desain Glassmorphism:** Tampilan elegan bergaya kaca.
- **Responsif 100%:** Optimal di Mobile, Tablet, Laptop, dan Desktop.
- **Performa Tinggi:** HTML, CSS, dan JS murni (Vanilla) tanpa library/framework besar, memastikan loading cepat.
- **SEO Optimized:** Penggunaan tag semantik HTML5 dan meta tag yang tepat.

## Cara Menjalankan Secara Lokal
1. Pastikan Anda memiliki browser modern (Chrome, Firefox, Safari, Edge).
2. Buka file `index.html` langsung di browser Anda.
3. Atau, gunakan Live Server di VS Code untuk pengalaman development yang lebih baik.

## Deployment ke Vercel / GitHub Pages
Website ini sudah dioptimasi dan siap di-deploy secara langsung tanpa proses *build* atau konfigurasi tambahan.

1. Upload seluruh file dan folder ke repository GitHub.
2. Di Vercel, buat proyek baru dan hubungkan dengan repository GitHub Anda.
3. Vercel akan secara otomatis mendeteksi file `index.html` dan melakukan *deploy*.
4. Website akan langsung bisa diakses secara publik.
